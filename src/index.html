What is it?

Shappd is a design tool that helps you craft unique and good looking custom shapes with the CSS border property. This tool is a good fit when you need blob shapes for your elements.

How does Blobmaker work?

You can create your custom shapes by moving the sliders —on the right— around until you find a shape that you like. And then copy the property values below using the copy button or ctrl + c

Can I download the design in svg or png format?

No!, but what you can do is to copy the property values and paste it in your code where you need that shape.
However, you can try other tools like blobmaker which lets you export SVG or PNG images of the shapes you make.
